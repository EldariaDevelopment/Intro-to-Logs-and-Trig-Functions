# Logs
## Introduction to Logs
Logs are the inverse of exponents, we use the notation of $\log_a$ to represent logs. 
## Log usage
How we use logs is that when we have spome $a^b$ where $b$ is the exponent to some a, some $\log_a a^b = b$
